# prompt-engineering-guide

## SET CLEAR GOALS
  - Use action goals
  - Define length & format
  - Specify audience

## PROVIDE CONTEXTUAL INFORMATION
  - Include relevant informations
  - Handing documents
  - Define key terms and concepts

## USE N-SHOT PROMPTING
  - (input) -> (output) examples
  - Style/tone examples
  - Examples of different levels of completions

## SPECIFICITY
  - Precise language/Straight to the point
  - Quantify request
  - Break down big concept into smaller ones

## EXPERIMENT AND PLAY AROUND
  - Different key words or formulations
  - Different level of detail
  - Different query length
  - 
## RELY ON CoT (chain of thought)
  - Encourage step by step reasoning
  - Ask model to explain reasoning process
  - Guide model through a sequence of logical thoughts


Source: [Google: What is prompt engineering](https://cloud.google.com/discover/what-is-prompt-engineering)
